# 手势识别控制方案

> 状态：方案已完成，Python 脚本已写好，待明天实测

## 架构

```
摄像头 ──▶ PC (Python) ──▶ 蓝牙 COM 口 ──▶ HC-05 ──▶ STM32 ──▶ OLED
              │
        MediaPipe Hands
        21个手部关键点
         → 5种手势分类
         → 发送单字符
```

**STM32 侧零改动**，HC-05 已支持单字符命令（t/f/z/s/h）。

---

## 环境搭建

```bash
pip install mediapipe opencv-python pyserial
```

PC 蓝牙配对 HC-05（密码 `1234`）→ 自动生成虚拟 COM 口。

---

## 手势 → 命令映射

| 手势 | 手势名称 | 发送字符 | 宠物反应 |
|---|---|---|---|
| ✋ 五指张开 | OPEN_PALM | `t` | SURPRISE 摸头 |
| 👍 竖起拇指 | THUMB_UP | `f` | HAPPY 喂食 |
| ✊ 握拳 | FIST | `z` | SLEEP 睡觉 |
| ☝️ 食指向上 | POINT_UP | `s` | 查询状态 |
| ✌️ V 手势 | VICTORY | `h` | 帮助菜单 |

---

## 手势识别原理

MediaPipe 返回 21 个手部关键点坐标：

```
手指伸直判断: 指尖 y 坐标 < PIP 关节 y 坐标
手指弯曲判断: 指尖 y 坐标 > PIP 关节 y 坐标
```

| 手势 | 判断逻辑 |
|---|---|
| OPEN_PALM | 4 指以上伸直 |
| FIST | 5 指全部弯曲 |
| THUMB_UP | 拇指翘起 + 其余 3 指弯曲 |
| POINT_UP | 仅食指伸直 |
| VICTORY | 食指+中指伸直，无名指+小指弯曲 |

---

## 防抖机制

- **帧确认**: 连续 5 帧相同手势才触发
- **冷却时间**: 两次发送间隔 ≥ 1.5 秒
- **单手势处理**: max_num_hands=1，只跟踪一只手

## 运行方式

```bash
# 自动查找 HC-05 COM 口
python Tools/gesture_control.py

# 指定 COM 口
python Tools/gesture_control.py COM5

# 不接蓝牙也能跑 (仅预览手势,不发送)
python Tools/gesture_control.py
```
